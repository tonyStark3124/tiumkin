# Bosserha — בוסר המלאכים

Landing page for Rabbi Uri Zohar & Dan Tiomkin's parenting courses — practical guidance for parents of at-risk youth and children with ADHD.

Built and deployed for a real client. Live at [bosserha.com](https://bosserha.com).

## What it does

- Presents course offerings and instructor credentials
- Drives conversions via integrated payment flow
- Mobile-first, Hebrew RTL, optimized for the target demographic (Israeli parents, 35–60)

## Tech

| | |
|---|---|
| Framework | React (CRA) |
| Language | TypeScript |
| Routing | React Router |
| Styling | CSS Modules |
| Deploy | Netlify |
